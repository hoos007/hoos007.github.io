## 녹화기능 구현
### gui 구현
파이썬으로 gui 프로그램을 한번도 안만들어 봐서 chat gpt한테 물어봤습니다. (gpt-3.5)
그랬더니

> 파이썬으로 GUI 프로그램을 개발하기 위한 여러 가지 라이브러리가 있습니다. 여기에는 Tkinter, PyQt, wxPython, Kivy 등이 있습니다. 각각의 라이브러리는 강점과 특징을 가지고 있으므로, 프로젝트의 요구 사항과 개발자의 경험에 따라 선택할 수 있습니다.
Tkinter: 파이썬의 기본 GUI 라이브러리로써 가벼우며 사용하기 쉽습니다. 표준 라이브러리로 제공되므로 설치할 필요가 없습니다.
PyQt: Qt 프레임워크를 파이썬으로 사용할 수 있도록 하는 라이브러리입니다. Qt는 강력하고 다양한 기능을 제공하며, PyQt는 Qt에 대한 파이썬 바인딩을 제공합니다.
wxPython: 크로스 플랫폼 GUI 개발을 위한 라이브러리입니다. C++로 작성된 wxWidgets 라이브러리의 파이썬 바인딩을 제공합니다.
Kivy: 멀티 플랫폼 애플리케이션을 개발하기 위한 라이브러리로써, 터치 기반 인터페이스에 중점을 둡니다. Kivy는 사용자 인터페이스와 관련된 다양한 기능을 제공합니다.
이 외에도 PySide, PyGTK, wxWidgets 등 다른 라이브러리도 있습니다. 각 라이브러리는 자체 문서와 예제를 제공하므로, 프로젝트에 맞는 라이브러리를 선택한 후 해당 라이브러리의 문서와 예제를 참고하여 개발을 진행할 수 있습니다.

Tkinter, PyQt, wxPython, Kivy 를 추천받았습니다.
그래서 간단한거 만들생각인데 뭐가 좋을까 물어봤습니다.

> 간단한 GUI 프로그램을 만들기 위해선 Tkinter가 가장 적합한 선택일 것입니다. Tkinter는 파이썬의 표준 라이브러리이기 때문에 설치할 필요 없이 파이썬과 함께 제공됩니다.

간단한거 만들기로는 Tkinter가 좋다고 하네요.
사실 제가 만들 프로젝트가 간단한지 어떤지는 확실하게는 모르겠습니다. 해보고 안되겠다싶으면 그때가서 새로 만들 각오로 일단 Tkinter로 해보기로 결정했습니다.

사실 여기까지 물어보고 일단 gui 만들어야 겠다고 생각했습니다만 문서를 읽다가 조금 귀찮아져서 gpt한테 먼저 물어보기로 했습니다. 안돼면 직접하고 잘 만들어주면 그거 써야지~ 라고 생각했습니다.

먼저 구체적으로 내가 만들고싶은게 뭔지 가르쳐주고 어떤 레이아웃을 원하는지 자세하게 설명했습니다. 처음에는 의도를 벗어나는 경우도 있었는데 계속 이거 아니라고 수정해주니까 어느정도 모양이 나왔습니다.
gpt가 만든 결과물입니다.
```python
import tkinter
import tkinter.ttk
from tkinter import messagebox

streamTitle = "test"
streamDate = "test"
streamTime = "test"
streamTheme = "test"

def toggle_recording():
    # 녹화 대기 상태를 토글하는 함수
    global is_recording
    if is_recording:
        result = messagebox.askokcancel("녹화 종료 확인", "녹화중인 상태인 경우 녹화가 강제 종료됩니다.")
        if result:
            # 녹화 종료 처리 추가
            is_recording = False
            record_button.config(text="녹화대기 켜기", bg="white")
            status_label.config(text="현재상태: 자동 녹화 대기중 아님")
    else:
        is_recording = not is_recording
        record_button.config(text="녹화대기 끄기", bg="red")
        status_label.config(text="현재상태: 자동 녹화 대기중")

root = tkinter.Tk()
root.title("Twitch Clip Helper")
root.geometry("1280x720")  # 창 크기 조정
root.resizable(False, False)  # 창 크기 고정

notebook = tkinter.ttk.Notebook(root, width=1280, height=720)  # 노트북 크기 조정
notebook.pack()

tab1 = tkinter.Frame(root)
notebook.add(tab1, text="녹화")
tab2 = tkinter.Frame(root)
notebook.add(tab2, text="클립편집")

# 녹화 탭 UI
search_frame = tkinter.Frame(tab1)
search_frame.pack(pady=20)

search_entry = tkinter.Entry(search_frame, width=40)  # 텍스트 박스 크기 조정
search_entry.pack(side=tkinter.LEFT, padx=5)

search_button = tkinter.Button(search_frame, text="검색")
search_button.pack(side=tkinter.LEFT)

parent_frame = tkinter.Frame(tab1)
parent_frame.pack(pady=20)

preview_frame = tkinter.Frame(parent_frame)
preview_frame.grid(row=0, column=0, padx=50, sticky=tkinter.N)

preview_canvas = tkinter.Canvas(preview_frame, width=360, height=202, bg="black")  # 미리보기 영상 크기 조정 및 검정색 배경 설정
preview_canvas.pack()

info_frame = tkinter.LabelFrame(parent_frame, text="방송 정보", font=("Arial", 16), padx=50, pady=50)  # 영상 정보 라벨 프레임 생성
info_frame.grid(row=0, column=1, padx=50, sticky=tkinter.N)

info_label = tkinter.Label(info_frame, text="방송 제목 : " + streamTitle + "\n"
                                            "방송 일자 : " + streamDate + "\n"
                                            "방송 시간 : " + streamTime + "\n"
                                            "방송 테마 : " + streamTheme, font=("Arial", 15))  # 영상 제목 라벨 중앙 정렬
info_label.pack()

button_label_frame = tkinter.Frame(tab1)  # 버튼과 라벨을 포함할 프레임 생성
button_label_frame.pack(pady=20)

is_recording = False
record_button = tkinter.Button(button_label_frame, text="녹화대기 켜기", command=toggle_recording, font=("Arial", 12), bg="white")  # 녹화 버튼 폰트 크기 조정
record_button.pack(side=tkinter.LEFT, padx=10)

status_label = tkinter.Label(button_label_frame, text="현재상태: 자동 녹화 대기중 아님", font=("Arial", 14))  # 상태 표시 라벨 추가
status_label.pack(side=tkinter.LEFT)

progressbar_frame = tkinter.Frame(tab1)  # 프레임 생성
progressbar_frame.pack(pady=20)

progressbar = tkinter.ttk.Progressbar(progressbar_frame, mode="indeterminate", length=400)  # 프로그레스바 생성
progressbar.pack()
progressbar.start()

root.mainloop()

```
