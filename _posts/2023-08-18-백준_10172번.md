### 제출
```java
public class Main{
    public static void main(String[] args)
    {
        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\\\__|");
    }
}
```

### 풀이
주의해야할 점은 역슬래시와 쌍따옴표가 출력되지 않는다는 것이다.

역슬래시를 출력하기 위해서는 `\\`를 사용해야 한다.

쌍따옴표를 출력하기 위해서는 `\"`를 사용해야 한다.

```java
System.out.println("|\\_/|");
System.out.println("|q p|   /}");
System.out.println("( 0 )\"\"\"\\");
System.out.println("|\"^\"`    |");
System.out.println("||_/=\\\\__|");
```
이를 통해 출력이 가능하다.

`println()`함수를 통해 출력을 할 수 있다. 이 함수는 줄 내림을 포함하고있다.

`print()`함수를 통해 출력을 할 수 있다. 이 함수는 줄 내림을 포함하지 않는다.

### 개념 정리
#### escape sequence(이스케이프 시퀀스, 제어 문자)

| 제어 문자 | 설명                              |
|----------|---------------------------------|
| `\t`     | 탭(tab)                        |
| `\b`     | 백스페이스(backspace)           |
| `\n`     | 새 줄(new line)                 |
| `\r`     | 캐리지 리턴(carriage return)    |
| `\f`     | 폼 피드(form feed)              |
| `\'`     | 단일 인용부호(single quote)     |
| `\"`     | 더블 인용부호(double quote)     |
| `\\`     | 백슬래시(backslash)              |
| `\0`     | Null 문자                      |
| `\uXXXX` | 유니코드 문자(XXXX는 16진수)     |
| `\ooo`   | 8진수(oct)로 표현된 문자          |

이러한 이스케이프 시퀀스는 문자열 내에서 특별한 문자나 특정 형식의 문자를 표현하기 위해 사용됩니다.
