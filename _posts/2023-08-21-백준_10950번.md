### 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

### 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다.

각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

### 출력
각 테스트 케이스마다 A+B를 출력한다.

### 예제 입력 1
> 5
> <br>1 1
> <br>2 3
> <br>3 4
> <br>9 8
> <br>5 2

### 예제 출력 1
> 2
> <br>5
> <br>7
> <br>17
> <br>7

### 알고리즘 분류
- 수학
- 구현
- 사칙연산

***
### 제출
```java
import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner stdin = new Scanner(System.in);
        int cnt = stdin.nextInt();
        int n1;
        int n2;

        for(int i=0;i<cnt;i++){
            n1 = stdin.nextInt();
            n2 = stdin.nextInt();
            System.out.println(n1+n2);
        }
    }
}
```

### 풀이
반복문을 통해 결과가 나오도록 한다.

```java
for(int i=0;i<cnt;i++){
    n1 = stdin.nextInt();
    n2 = stdin.nextInt();
    System.out.println(n1+n2);
}
```
for문을 사용해 케이스 수만큼 반복하도록 하고 각 사이클 마다 값을 입력받은뒤 계산값을 출력했다.

이를 통해 값의 출력이 가능하다.
