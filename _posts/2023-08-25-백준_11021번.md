### 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

### 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다.

각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

### 출력
각 테스트 케이스마다 "Case #x: "를 출력한 다음, A+B를 출력한다. 테스트 케이스 번호는 1부터 시작한다.

### 예제 입력 1
> 5
> <br>1 1
> <br>2 3
> <br>3 4
> <br>9 8
> <br>5 2

### 예제 출력 1
> Case #1: 2
> <br>Case #2: 5
> <br>Case #3: 7
> <br>Case #4: 17
> <br>Case #5: 7

### 제출
```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.StringTokenizer;

public class Main{
    public static void main(String[] args) throws NumberFormatException, IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        int cnt = Integer.parseInt(br.readLine());
        StringTokenizer st;

        for (int i=1;i<=cnt;i++)
        {
            st = new StringTokenizer(br.readLine());
            bw.write("Case #"+i+": "+(Integer.parseInt(st.nextToken())+Integer.parseInt(st.nextToken()))+"\n");
        }
        bw.flush();
        bw.close();
    }
}
```

### 풀이
이번 문제는 BufferedReader와 BufferedWriter에 익숙해지고 싶어서 Scanner대신 이를 할용해 해결했다.

이 클래스는 readLine()메서드만 제공하고 정규식 처리 없이 단순하게 문자열을 읽기만 하도록 내부적으로 동작하기 때문에 속도측면에서 Scanner보다 더 빠르다.

InputStreamReader를 사용하는 방법도 있지만 이번에는 BufferedReader를 사용해서 문제를 해결했다.

사용법은 다음과 같다.

```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
```

입력을 위한 BufferedReader와 출력을 위한 BufferedWriter 객체를 생성한다.

InputStreamReader는 바이트 기반의 입력 스트림을 문자 스트림으로 변환하는 클래스이고 System.in은 표준 입력 스트림으로 바이트 기반이다.
<br>System.in을 통해 바이트 데이터를 읽고 InputStreamReader를 통해 문자로 변환한 후에 이것을 BufferedReader가 받는 것이다.

OutputStreamWriter는 문자기반의 출력을 바이트 기반의 출력 스트림으로 변환하는 클래스이고 System.out은 표준 출력 스크림으로 바이트 기반이다.

그 다음으로 BufferdReader는 라인 단위로 문자열을 가져오기 때문에 분리가 필요하다면 StringTokenizer를 통해 분리하는 과정이 필요하다.

```java
StringTokenizer st; // 변수 생성

st = new StringTokenizer(br.readLine()); // 객체 생성 

Integer.parseInt(st.nextToken()); // nextToken을 나눠진 Token단위로 하나씩 가져올 수 있음.
```
이를 통해 입력에 가능하다.

출력은 write()메서드를 통해 가능하다.

```java
bw.write("Case #"+i+": "+(Integer.parseInt(st.nextToken())+Integer.parseInt(st.nextToken()))+"\n");

bw.flush(); // 강제 버퍼 비우기
bw.close(); // BuffuredWriter 닫기
```
주의할 점은 버퍼를 사용하면 비워야 한다는 것 인데 혹시나 버퍼에 남아있는 내용이 있을때 다음 출력에서 그 내용들이 같이 출력되는 문제를 방지하기 위함이다.

이를 통해 원하는 값의 출력을 Scanner를 사용할때보다 짧은 시간에 가능하다.
