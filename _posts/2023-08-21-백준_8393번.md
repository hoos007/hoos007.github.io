### 문제
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

### 입력
첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

### 출력
1부터 n까지 합을 출력한다.

### 예제 입력 1
> 3

### 예제 출력 1
> 6

### 알고리즘 분류
- 수학
- 구현

***
### 제출
```java
import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner stdin = new Scanner(System.in);
        int n = stdin.nextInt();
        int result = 0;

        for(int i=1;i<=n;i++){
            result += i;
        }
        System.out.print(result);
    }
}
```

### 풀이
반복문을 통해 결과가 나오도록 한다.

```java
for(int i=1;i<=n;i++){
    result += i;
}
System.out.print(result);
```
for문을 사용해 1부터 n까지 반복하도록 하고 각 사이클 마다 값을 더하고 끝난후에 출력했다.

이를 통해 값의 출력이 가능하다.
